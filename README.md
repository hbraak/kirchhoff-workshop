# Kirchhoff-Workshop

Interaktiver HTML-Workshop zum Thema **Widerstandsnetzwerke** für Berufskolleg-Schüler (11. Klasse Physiktechnik).

## Themen
1. Parallelschaltung
2. Gemischte Schaltungen
3. Wheatstone-Brücke
4. Kirchhoff'sche Regeln
5. Gleichungssystem aufstellen & lösen
6. Freier Rechner

## Verwendung
Einfach `index.html` im Browser öffnen. Keine Installation, keine Dependencies, funktioniert offline.

**Live:** [https://hbraak.github.io/kirchhoff-workshop/](https://hbraak.github.io/kirchhoff-workshop/)

## Kontext
- Lore-Lorentz-Schule Düsseldorf
- Default-Werte: R1=100Ω, R2=220Ω, R3=330Ω, R4=470Ω, R5=150Ω, U0=12V
- Lehrer: Heiko Braak
